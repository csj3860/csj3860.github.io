Branching Point

How to find the Branching Point?

1. Introduction to Branching Point

A common problem in Multi-valued Functions is to get the branching points.

What is branching points? If the variables walk on a path which surrounds single one branching point, the multi-valued function value will change.

For example, root function:
$$
\begin{align*}
w=\sqrt{z-a}
\end{align*}
$$
The multi-valued property comes from $z-a$, especially its phase.

Let’s consider this circumstance: A close path without surrounding $z=a$, then we can see that the phase of $z-a$ remain unchanged here so the value does not change.

But things are different when the path surrounds some points.

One of branching points is $a$, because a close path surrounding $z=a$ will change $2\pi$ in phase of $\sqrt{z-a}$ so that the result will change $\pi$ in phase. But here we should be careful that if the original phase changes $4\pi$ then the final result will not change indeed as $2\pi$ is phase period.

What’s more, our friend $\infty$ maybe the branching point. If we take account for a huge circle, then it will surround all points including $z=a$, so it is nature that the result will change.

Then how to find branching point in other more complicated functions like $\sqrt[3]{z-a}$ and $\sqrt{(z-a)(z-b)}$. And is the $\infty$ one of branching points?

2. Find and Verify the Branching Point in Root Function

For finite branching points, our task is to find them while verifying it in terms of $\infty$.

In the beginning, just think about one function : $w=\sqrt{z}$.

The branching point is very clear, $z=0$.

In general situation, function $w=\sqrt[n]{z}(n>2,n \in Z)$ , the first branching point must be zero.

Develop further, we can say that single roots of $p(z)=0(w=\sqrt[n]{p(z)})$ belong to branching points

Method 1(Find or Solve the Equation): Single roots are all branching points.

In this way, all finite branching points can be known.

So how to verify whether $\infty$ is branching point or not? We can draw a close curve, maybe not so small, to surround all finite branching points, and then let variable $z$ walk on this curve. In the end we will find that the phase change largely, not just $2\pi$ but $Number(\text{finite branching points})\times 2 \pi$. Then we recalculate the final phase changing to see whether it is multiple of $2\pi$.

Method 2(Verify the Quality of $\infty$): Surround all finite branching points and calculate the total results.
